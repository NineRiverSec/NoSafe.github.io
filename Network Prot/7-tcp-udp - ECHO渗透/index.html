  • 基本信息

该主机上正在运行一个echo服务。echo服务用于测试和测量目的，可以监听TCP和UDP协议。服务器发送回它接收到的任何数据，不做任何修改。

通过将回声服务连接到同一台或另一台机器上的回声服务，可能会导致拒绝服务。由于产生的数据包数量过高，受影响的机器可能会有效地停止服务。

参考 https://www.acunetix.com/vulnerabilities/web/echo-service-running/

默认端口: 7/tcp/udp

PORT STATE SERVICE 7/udp open echo 7/tcp open echo

## 访问 Echo 服务(UDP)

nc -uvn <IP> 7 Hello echo #This is wat you send Hello echo #This is the response
